# Enrutador

Proyecto básico en React + Vite con páginas visuales simples.

## Páginas creadas

- Services
- Productos
- Contact

Cada página está hecha como componente independiente en `src/pages` con su archivo CSS.

## Ejecutar el proyecto

1. Instalar dependencias:

```bash
npm install
```

2. Iniciar en desarrollo:

```bash
npm run dev
```
